# DataChallenges:

A data challenge is:
- a specification of the task:
  - input quantities / spatial and temporal domains
  - target quantities / spatial and temporal domains
  - offlimit domains for method development
- a collection of configured pipelines for:
  - loading data
  - computing metrics
- Versioned:
  - reference and inference datasets
  - pipeline src
  - runtime environment
- A automated tool for submiting new results
